# TC-001. Успешная авторизация

ID: TC-001
Приоритет: High

Предусловия:
- Пользователь находится на странице авторизации.

## Шаги

1. Ввести корректный логин.
2. Ввести корректный пароль.
3. Нажать кнопку Login.

## Ожидаемый результат

Пользователь успешно авторизован и открывается каталог товаров.

# TC-007. Авторизация заблокированного пользователя

ID: TC-007
Приоритет: High

Предусловия:
- Пользователь находится на странице авторизации.

## Шаги

1. Ввести логин locked_out_user.
2. Ввести корректный пароль.
3. Нажать кнопку Login.

## Ожидаемый результат

Отображается сообщение о том, что пользователь заблокирован. Авторизация не выполняется.
