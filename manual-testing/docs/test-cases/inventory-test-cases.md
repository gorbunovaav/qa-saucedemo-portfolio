# TC-020. Открытие каталога товаров после авторизации

ID: TC-020  
Приоритет: High

Предусловия:
- Пользователь находится на странице логина.
- Используются валидные данные пользователя.

## Шаги

1. Ввести логин `standard_user`.
2. Ввести пароль `secret_sauce`.
3. Нажать кнопку Login.

## Ожидаемый результат

Пользователь успешно авторизуется и попадает на страницу каталога товаров.
