# BUG-004

## Название

Нажатие Enter на странице Checkout возвращает пользователя в корзину вместо отправки формы

## Severity

Major

## Priority

Medium

## Окружение

- macOS
- Google Chrome (последняя версия)

## Предусловия

Открыта страница оформления заказа (Checkout).

## Шаги воспроизведения

1. Добавить товар в корзину.
2. Перейти на страницу Checkout.
3. Заполнить обязательные поля корректными данными.
4. Нажать Enter.

## Ожидаемый результат

Форма обрабатывается так же, как при нажатии кнопки Continue: пользователь переходит на следующий шаг оформления заказа.

## Фактический результат

Пользователь возвращается на страницу корзины (Cart), введенные данные теряются, переход на следующий шаг оформления заказа не выполняется.
