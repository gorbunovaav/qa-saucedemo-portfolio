# Checklist. Корзина

## Добавление товаров

- [ ] Добавление одного товара
- [ ] Добавление нескольких разных товаров
- [ ] Добавление всех товаров
- [ ] Один товар нельзя добавить дважды
- [ ] После добавления меняется кнопка "Add to cart" → "Remove"
- [ ] Счетчик корзины увеличивается

## Удаление товаров

- [ ] Удаление одного товара
- [ ] Удаление нескольких товаров
- [ ] Удаление всех товаров
- [ ] После удаления кнопка становится "Add to cart"
- [ ] Счетчик корзины уменьшается
- [ ] После удаления последнего товара счетчик исчезает

## Корзина

- [ ] Открывается корзина
- [ ] Отображаются все добавленные товары
- [ ] Отображается название товара
- [ ] Отображается описание товара
- [ ] Отображается цена
- [ ] Отображается количество товаров

## Навигация

- [ ] Работает кнопка Continue Shopping
- [ ] Работает кнопка Checkout

## Сохранение состояния

- [ ] Корзина сохраняется после обновления страницы
- [ ] Корзина сохраняется после перехода между страницами
- [ ] Корзина очищается после завершения заказа

## Негативные проверки

- [ ] Нельзя перейти к оформлению заказа с пустой корзиной
- [ ] После Logout корзина очищается (если это предусмотрено требованиями)
