# Checklist. Авторизация

## Функциональное тестирование

- [ ] Успешный вход с корректными данными
- [ ] Вход с неверным паролем
- [ ] Вход с неверным логином
- [ ] Пустой логин
- [ ] Пустой пароль
- [ ] Пустые оба поля
- [ ] Проверка клавиши Enter
- [ ] Выход из аккаунта
- [ ] Прямой переход на защищённую страницу без авторизации
- [ ] Возврат назад после Logout

## UX

- [ ] Поле пароля скрывает символы
- [ ] Есть возможность показать пароль
- [ ] Сообщения об ошибках понятны пользователю
- [ ] Ошибка отображается полностью
- [ ] Кнопка Login доступна и заметна

## Безопасность

- [ ] Проверка SQL-инъекции
- [ ] Проверка XSS
- [ ] Ограничение количества попыток входа
- [ ] Безопасная передача данных (HTTPS)
- [ ] Проверка восстановления пароля (если предусмотрено)
