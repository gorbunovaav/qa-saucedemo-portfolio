# TC-0014. Успешное оформление заказа

ID: TC-014
Приоритет: High

Предусловия:
- Пользователь авторизован.
- В корзине находится хотя бы один товар.
- Открыта страница Checkout.

## Шаги

1. Заполнить поле First Name.
2. Заполнить поле Last Name.
3. Заполнить поле Zip Code.
4. Нажать кнопку Continue.

## Ожидаемый результат

Открывается страница подтверждения заказа.

# TC-015. Оформление заказа без имени

ID: TC-015
Приоритет: High

Предусловия:
- Пользователь находится на странице Checkout.

## Шаги

1. Оставить поле First Name пустым.
2. Заполнить остальные обязательные поля.
3. Нажать кнопку Continue.

## Ожидаемый результат

Отображается сообщение об ошибке валидации.

# TC-018. Завершение оформления заказа

ID: TC-018
Приоритет: High

Предусловия:
- Пользователь находится на странице подтверждения заказа (Checkout Overview).

## Шаги

1. Нажать кнопку Finish.

## Ожидаемый результат

Заказ успешно оформляется, отображается сообщение об успешном завершении покупки.

# TC-019. Возврат в каталог после оформления заказа

ID: TC-019
Приоритет: Medium

Предусловия:
- Заказ успешно оформлен.

## Шаги

1. Нажать кнопку Back Home.

## Ожидаемый результат

Пользователь возвращается в каталог товаров.
