# TC-008. Добавление товара в корзину

ID: TC-008
Приоритет: High

Предусловия:
- Пользователь авторизован.
- Открыт каталог товаров.

## Шаги

1. Нажать кнопку Add to cart у любого товара.

## Ожидаемый результат

Товар добавляется в корзину, кнопка меняется на Remove, счетчик корзины увеличивается на 1.

---

# TC-009. Добавление нескольких товаров в корзину

ID: TC-009
Приоритет: High

Предусловия:
- Пользователь авторизован.
- Открыт каталог товаров.

## Шаги

1. Добавить первый товар в корзину.
2. Добавить второй товар в корзину.
3. Открыть корзину.

## Ожидаемый результат

В корзине отображаются оба выбранных товара, счетчик корзины показывает количество добавленных товаров.

---

# TC-010. Удаление товара из корзины

ID: TC-010
Приоритет: High

Предусловия:
- Пользователь авторизован.
- В корзине находится хотя бы один товар.

## Шаги

1. Нажать кнопку Remove у товара.

## Ожидаемый результат

Товар удаляется из корзины, счетчик корзины уменьшается.
