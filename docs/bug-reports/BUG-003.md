# BUG-003

## Название

Некорректное отображение иконки ошибки при вводе длинной строки

## Severity

Minor

## Priority

Medium

## Окружение

- Windows 11
- Google Chrome (последняя версия)

## Предусловия

Открыта страница авторизации.

## Шаги воспроизведения

1. Открыть страницу авторизации.
2. Ввести длинный некорректный логин.
3. Нажать **Login**.

## Ожидаемый результат

Иконка ошибки отображается корректно и не просвечивает.

## Фактический результат

Через белую область иконки просвечивает введенный текст.
