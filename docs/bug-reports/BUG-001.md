# BUG-001

## Название

Сообщение об ошибке отображается не полностью на странице авторизации

## Severity

Minor

## Priority

Medium

## Окружение

- macOS Sequoia 15.x
- Google Chrome (последняя версия)

## Предусловия

Открыта страница авторизации.

## Шаги воспроизведения

1. Открыть страницу авторизации.
2. Ввести неверный логин или пароль.
3. Нажать кнопку **Login**.

## Ожидаемый результат

Сообщение об ошибке отображается полностью и читается без затруднений.

## Фактический результат

Текст сообщения об ошибке обрезается и отображается не полностью.
