# BUG-002

## Название

Некорректная обработка клавиши Enter после неуспешной авторизации

## Severity

Minor

## Priority

Medium

## Окружение

- macOS
- Google Chrome (последняя версия)

## Предусловия

Открыта страница авторизации.

## Шаги воспроизведения

1. Открыть страницу авторизации.
2. Ввести неверный логин и/или пароль.
3. Нажать **Login**.
4. Дождаться появления сообщения об ошибке.
5. Нажать **Enter**.
6. Ввести корректные данные.
7. Нажать **Enter** один раз.
8. Нажать **Enter** второй раз.

## Ожидаемый результат

- После появления ошибки нажатие **Enter** не должно изменять состояние страницы до новой попытки авторизации.
- После ввода корректных данных вход должен выполняться после первого нажатия **Enter**.

## Фактический результат

- После появления ошибки нажатие **Enter** скрывает сообщение об ошибке.
- После ввода корректных данных первое нажатие **Enter** не выполняет вход. Авторизация происходит только после повторного нажатия **Enter**.
