# SauceDemo QA Portfolio

Проект демонстрирует навыки ручного и автоматизированного тестирования интернет-магазина SauceDemo.

## Что реализовано

### Manual Testing

- Smoke Checklist
- Login Checklist
- Cart Checklist
- Checkout Checklist

- Test Cases
- Bug Reports

### Automation

- Playwright
- TypeScript
- Page Object Model
- Chrome
- Firefox
- Safari

## Структура проекта

manual-testing/
playwright/

## Запуск

cd playwright

npm install

npx playwright test

## Отчет

npx playwright show-report
